# JonesEngineeringINFX435

## Prerequisites

Oracle19c

Python >=3.7

Pip >=9.0.3

## Setup

Run jones_engineering.sql 

Change config.py to your Oracle19c configuration 

`python3 -m pip install -r requirements.txt`

`export FLASK_APP=app`


## Start app

`python3 -m flask run`

Navigate to http://localhost:5000/home
